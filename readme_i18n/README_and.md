## 🥛 Sobre er Proyêtto
**Kéfî Contrôh** naçió de la neçeçidá de recordâh cuándo tu fermentaçión êttá en çu punto perfêtto. Ya çea kéfî de lexe o kombuxa, êtta aplicaçión çimplifica er proçeço con notificaçionê localê programâh, un temporiçadôh en bibo y aprendiçahe baçao en tû gûttô.

Er proyêtto êh **100% Free and Open Source Software (FOSS)**, çentrao en la pribaçidá y con una interfâh modênna baçá en Material Design 3.

## ✨ Principale Caracteríttica
- **🥛 Hêttión de Kéfî y Kombuxa**: Çoporte êppeçífico pa diferentê tipô de fermentaçión con etapâ perçonaliçâh.
- **⏱️ Tiempô Ideâlê Intelîhentê**: La âpp aprende de tû coçexâ paçâh pa çuherirte er tiempo de fermentaçión que mâh te gûtta.
- **🔔 Notificaçionê y Pre-abiçô**: Alarmâ localê (çin intênnêh) que te abiçan ar finaliçâh y 2 orâ antê.
- **♾️ Modo Libre**: Iniçia fermentaçionê çin límite de tiempo pa un contrôh totarmente manuâh.
- **📅 Calendario Integrao**: Biçualiça tu îttoriâh y planifica futurâ tandâ de forma biçuâh.
- **📱 Material You**: Çoporte pa colorê dinámicô y tema ôccuro/claro fiêh a Material Design 3.
- **📳 Feedback Háttico**: Interâççionê fîçicâ mediante bibraçión pa una êpperiençia mâh îmmerçiba.
- **💾 Copiâ de Çeguridad**: Êpportaçión e împortaçión de datô en formato JSON.
- **🌍 Multilingüe**: Dîpponible en Câtteyano, Ingléh y L'Andalú (EPA).
- **🔒 Pribaçidá Primero**: Çin cuentâ, çin râtteadorê y çin analíticâ. Tû datô çon çolo tuyô.

## 🛠️ Tecnología y Requîçitô
- [Flutter SDK](https://flutter.dev/) (>= 3.0.0)
- Paquetê clabe uçáô:
  - `shared_preferences` (Perçîttençia local)
  - `flutter_local_notifications` (Notificadore Natibô)
  - `flutter_timezone` (Hêttión de fexa y ora der çîttema)
  - `flex_color_scheme` (Temâ Material 3 abançáô)

## 🚀 Înttalaçión y Compilaçión pa Deçarroyadorê
Çi quierê compilâh l'aplicaçión tú mîmmo dêdde er código fuente:

1. Clona êtte repoçitorio:
   ```bash
   git clone https://github.com/raulmoralesruiz/kefir-control.git
   ```
2. Ênttra en la carpeta der proyêtto:
   ```bash
   cd kefir-control
   ```
3. Bâhatte lâ dependençiâ:
   ```bash
   flutter pub get
   ```
4. Ehecuta l'aplicaçión en tu emuladôh o dîppoçitibo fîçico:
   ```bash
   flutter run
   ```
